Instruções de Uso:

Realizar o cadastro na página;

Realizar Login com os dados cadastrados no cadastro.

Perfil Aluno:

Assim que efetuado o login, será redirecionado para a tela turmas. Na tela turmas, caso não seja cadastrado na turma, terá que se cadastrar, com o código e senha fornecido pelo professor. Caso já tenha se cadastrado na turma, basta iniciar o quiz ou escolher as outras opções que o sistema oferece, como conferir os vídeos sobre Orientação a Objetos, verificar o seu rendimento, acessar os materias complementares. Poderá realizar o Quiz quantas vezes desejar, será salva somente a pontuação mais alta.

Perfil Professor:

Assim que efetuado o login, será redirecionado para a tela turmas. Na tela turmas, o professor terá o mesmo acesso que o perfil aluno, exceto na tela turmas, onde será possível o mesmo efetuar o cadastramento de uma e disponibilizar o código e a senha para seus alunos.

Qualquer dúvidas ou informações, basta enviar um e-mail para amandadyaz@gmail.com, com a sua solicitação.

Espero que o sistema o ajude ao longo do aprendizado de Programação Orientada á Objetos.
